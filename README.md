### How to run this project?

#### Set up the virtual environment:
``` 
python -m venv cs4250 
```
- For Windows User:
``` 
./cs4250/Scripts/activate
```

- For Mac Users:
```
source cs4250/bin/activate
```
#### Download all necessary python packages 
```
pip install -r requirements.txt
```
#### Running the project


```
python ./main.py
```
